Linux Operating System

Linux is an operating system just like windows and Mac Various variants - Ubuntu - Redhat - Amazon Linux We will be accessing Linux through the terminal via SSH ls - list directory (has other parameters) pwd - present working directory cd - change directory mkdir - create a directory cd ~ - change to home directory cd / - change to root directory cd .. - return one step in the directory touch - create a file (without opening an interface to edit) vi - edit an existing file OR if the file doesnt exist, it creates the file and allows you to edit cat - view contents of an existing file. sudo - grant administrative privilege adduser - create a new user su - switch user
